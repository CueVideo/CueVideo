01 — Method

From a support set to a working detector.

No annotation team, no per-frame masks. You define the action by example; the model handles generalization and keeps adapting online.

01

Support

A support set of 3–5 clips defines the action — no bounding boxes, no per-frame labels.

02

Adaptation

The model adapts to the new class in seconds via student–teacher distillation — generalizing the action, not memorizing frames.

03

Inference

It recognizes the action in unseen video and refines online from sparse human corrections.

02 — The platform

See the platform in action.

Label a few clips, watch recognition appear, and correct it in place — the model sharpens from sparse feedback, live in the product.

04 — Why continual learning

Continual, not periodic.

Train from scratch
—Label thousands of frames by hand
—Weeks of pipeline and GPU time per change
—The world drifts; the model goes stale
—Every new action restarts the cycle
